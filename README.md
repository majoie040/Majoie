# Majoie
AI To-Do App — Project Description

The AI To-Do App is a smart productivity web application built with HTML, CSS, and JavaScript.
It helps users organize their daily tasks, manage priorities, and improve productivity with AI-powered features.

Users can:

* add, edit, and delete tasks,
* mark tasks as completed,
* organize tasks by category or priority,
* save tasks using local storage,
* and receive smart suggestions from AI.

The app includes a clean and responsive user interface designed for both desktop and mobile devices.

Main Features

* Task creation and management
* Priority levels (High, Medium, Low)
* Due dates and reminders
* Dark mode toggle
* Local storage support
* AI-generated productivity tips
* AI task suggestions and planning assistance
* Search and filter functionality

Technologies Used

* HTML
* CSS
* JavaScript
* OpenAI API (for AI features)
